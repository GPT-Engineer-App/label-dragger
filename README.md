# label-dragger

Make an where the user can upload a floor plan (image) and then create labels that the user should be able to drag to desired location on top of the uploaded image. Make sure the image is displayed is large. Make sure the labels are in dark so that they are visible on top do white images.

## Setup

```sh
git clone https://github.com/GPT-Engineer-App/label-dragger.git
cd label-dragger
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
